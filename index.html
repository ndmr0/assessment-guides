UX Design Student Resources

Assessment Guides

A shared library of web-based assessment guides. Each assessment has its own page, folder, and public link.

A50200 User Experience Solutions - Assessment 1 Explore and validate the UX problem, then document prompt-assisted prototype development. A50201 Ideation, Innovation and Design - Assessment 1 Define a design challenge, research market needs, develop low-fidelity wireframes, and evaluate a UX concept. A50202 Interface Design and Development - Assessment 1 Explain UI design foundations, improve a mobile app prototype, and build a two-page website interface. A50203 Understanding User Experience (UX) - Assessment 1 Plan UX testing for Facebook and YouTube video features with research, personas, journey maps, hypotheses, and an action plan. A50206 Understanding the UX Workplace - Assessment 1 Build a UX partnership strategy for Uber driver experience, then present it professionally.
